# file_up_download_ui

主要练习一下用nodejs做后端，vue3写前端，完成登录注册功能，以及文件的上传与下载功能。

## Project Setup

```sh
npm install
```

### Compile and Hot-Reload for Development

```sh
npm run dev
```

### Type-Check, Compile and Minify for Production

```sh
npm run build
```

### Lint with [ESLint](https://eslint.org/)

```sh
npm run lint
```
